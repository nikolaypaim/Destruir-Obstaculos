app web que apresenta um jogo rapido de destruir obstaculos com o clique do mouse.

---
## 📚 Aprendizados
- loop for
- uso de funções
- eventos

## ⚙️ Funcionalidades
- clicar em todos os quadrados para vencer o jogo
- os quadrados são gerados em posições aleatórias

## 🛠️ Stack

- JavaScript
- HTML5
- CSS3

## 🚀 Deploy
1. Clone ou baixe este repositório.

```bash
git clone https://github.com/nikolaypaim/Destruir-Obstaculos
```

2. Abra o arquivo `index.html` em seu navegador.
